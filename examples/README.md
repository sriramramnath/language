# LevLang Examples

This directory contains simple games and blog posts about LevLang.

## Simple Games

Start with these examples to learn LevLang:

### 1. Hello World (`01_hello_world.lvl`)
The absolute simplest LevLang game. Shows text and a blue square on screen.

```bash
levlang run examples/01_hello_world.lvl
```

**Concepts:** Basic game structure, player object, UI text display

---

### 2. Moving Square (`02_moving_square.lvl`)
Control a blue square with arrow keys.

```bash
levlang run examples/02_moving_square.lvl
```

**Concepts:** Player movement, speed property

---

### 3. Two Squares (`03_two_squares.lvl`)
A controllable blue square and a static red square.

```bash
levlang run examples/03_two_squares.lvl
```

**Concepts:** Multiple objects (player + enemy)

---

### 4. Color Demo (`04_color_demo.lvl`)
Multiple colored objects demonstrating LevLang's color system.

```bash
levlang run examples/04_color_demo.lvl
```

**Concepts:** Different colors, multiple enemies

---

### 5. Fast Mover (`05_fast_mover.lvl`)
A fast-moving player with obstacles to avoid.

```bash
levlang run examples/05_fast_mover.lvl
```

**Concepts:** Different speeds, positioning
