# Change Log

All notable changes to the "levlang" extension will be documented in this file.

## [0.1.0] - 2025-11-09

### Added
- Initial release of LevLang syntax highlighting
- Support for .lvl file extension
- Syntax highlighting for keywords, strings, numbers, and comments
- Auto-closing pairs for brackets, quotes, and braces
- Comment toggling support (line and block comments)
- Code folding support
- Smart indentation rules
- Language configuration for better editing experience

### Supported Features
- Game declarations
- Player, enemy, road, ui, gameover blocks
- Property assignments
- Function calls
- String interpolation
- Numeric values with units (sec, ms)
- Boolean values
- Operators

### Keywords Highlighted
- Control flow: game, player, enemy, road, ui, gameover, sprite, scene
- Events: on, when, update, draw
- Conditionals: if, else, while, for, return
- Actions: start

### Future Plans
- IntelliSense/auto-completion
- Code snippets
- Error detection
- Integrated transpiler
- Live preview
