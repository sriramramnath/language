### Platform Support
- **macOS**: 14 or newer (Apple Silicon + Intel)
- **Windows**: 10 or newer
- **Linux**: Any distro with a stable Python 3 release
- **Dependencies**: Python 3.8+, pip, pygame (auto-installed)

### Steps
1. Clone the repo:
   ```bash
   git clone https://github.com/sriramramnath/language.git
   cd language
   ```
2. Install locally:
   ```bash
   pip install -e .
   ```
3. Verify:
   ```bash
   levlang --version
   ```
