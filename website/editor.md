# Editor Integration

## Integrated Fully in VS Code
- LevLang ships with a VS Code extension bundle (see `vscode-levlang/`).
- Provides syntax highlighting for both block syntax and component syntax.
- Snippets for common patterns (player blocks, spawners, UI overlays).
- Status bar commands for `levlang run` / `levlang watch`.

### Getting Started
1. Open the repo in VS Code.
2. Install the local extension: `code --install-extension vscode-levlang/levlang.vsix`.
3. Reload the window—`.lvl` files will highlight automatically.

> _Tip: combine the watch command with VS Code’s built-in terminal for a live feedback loop._
