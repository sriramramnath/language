# LevLang Website

This directory contains the official LevLang website.

## Viewing the Website

### Locally
Simply open `index.html` in your web browser:
```bash
open docs/index.html
```

### GitHub Pages
The website can be hosted on GitHub Pages at:
`https://sriramramnath.github.io/language/`

## Features

- **Hero Section**: LevLang branding with ASCII art
- **Download Section**: 
  - Download Source Code button (links to GitHub)
  - LevCode IDE "Coming Soon" card
- **Features Section**: Highlights of LevLang capabilities
- **Code Examples**: Side-by-side comparison of LevLang and generated Python
- **Getting Started**: Quick installation guide
- **Responsive Design**: Works on desktop and mobile

## Customization

The website is a single HTML file with embedded CSS. To customize:

1. Edit `index.html`
2. Modify colors in the CSS section (look for `#667eea` and `#764ba2` for the main gradient)
3. Update content in the HTML sections

## Deployment

To deploy to GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Set source to "main" branch and "/docs" folder
4. Save and wait for deployment

Your site will be live at: `https://[username].github.io/[repository]/`
